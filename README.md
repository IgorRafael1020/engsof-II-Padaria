# engsof-II-Padaria

## Resumo
#### Esse projeto tem como objetivo resolver os problemas de uma padaria fictícia chamada Cantinho do Trigo. Ela precisa de um sistema que controle o estoque das mercadorias e faça o controle das vendas.
